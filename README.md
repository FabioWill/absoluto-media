# Absoluto Media

Public brand media assets for **Absoluto.ai** — enterprise AI agent orchestration platform.

## Structure

```
.
├── logos/              # Official brand marks (SVG)
└── video/
    └── inputs/         # Pre-composed input frames for video generation (Kling, Veo, Seedance)
```

## Usage

Assets here are public by design. Any production system (CMS, video gen APIs, social media schedulers) can reference these via raw GitHub URLs.

Example raw URL:
```
https://raw.githubusercontent.com/FabioWill/absoluto-media/main/video/inputs/reels_simbolo_1080x1920.png
```
